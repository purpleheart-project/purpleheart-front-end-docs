# Writing Pre-request Scripts

## Scripts

Hoppscotch 允许您向REST 请求添加动态行为。
这允许您编写测试套件并构建可以包含动态参数的请求。
您可以添加javaScript在流程中的 2 个事件处执行的代码：

- 在向服务器发送请求之前，可以添加一个Pre-request 脚本
- 收到响应后，可以运行测试脚本。

>您可以将预请求和测试脚本添加到保存在集合中或未保存到集合中的请求。

然后，Hoppscotch 将按照指定的顺序执行脚本以及请求。

## 预请求脚本

预请求脚本是在请求执行之前运行的一段代码。

我们可以将预请求脚本用于预处理任务，例如：

- 设置参数、标题
- 添加正文数据

## 编写预请求脚本

一个特殊的`pw`API 对象包含用于创建脚本和测试的各种方法，并且可以在全球范围内使用。可以通过名称引用它来访问方法，例如

```javascript
pw.env.set("baseURL", "https://httpbin.org");
pw.env.set("method", "get");
```
